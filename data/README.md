# Dataset Description

This folder contains the datasets used in this project, collected from a university survey among undergraduate students, Faculty of Science and Technology, Universiti Kebangsaan Malaysia.

## Files

### 1. Demography.csv
Demographic information of the study participants, including basic background variables such as:

- **Height**
- **Weight**
- **BMI** 
- **Gender**  
- **Age**
- **Race**
- **Year of study**
- **Department**

### 2. Psychological_aspects.csv
Psychological-related variables collected from the same study, where:

- **M1 – M9**: Questionnaire questions for *Depression*, based on Patient Health Questionnaire-9 (PHQ-9)
- **R1 – R7**: Questionnaire questions for *Anxiety*, based on General Anxiety Disorder-7 (GAD-7)
- **T1 – T10**: Questionnaire questions for *Stress*, based on Perceived Stress Scale-10 (PSS-10)

## Notes
- Both datasets are used for analysis in the R scripts.
- Data have been cleaned and anonymized before analysis.
- No personally identifiable information is included.
