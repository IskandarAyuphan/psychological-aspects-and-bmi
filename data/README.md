# Dataset Description

This folder contains the datasets used in this project.

## Files

### 1. demography.csv
Demographic information of the study participants, including basic background variables such as:

- **Tinggi**: Height  
- **Berat**: Weight  
- **BMI**: Body Mass Index  
- **Jantina**: Gender  
- **Umur**: Age  
- **Bangsa**: Race  
- **Tahun**: Year of study  
- **Jabatan**: Department  

### 2. psychological_aspects.csv
Psychological-related variables collected from the same study, where:

- **M1 – M9**: Survey questions for *Kemurungan* (Depression)  
- **R1 – R7**: Survey questions for *Keresahan* (Anxiety)  
- **T1 – T10**: Survey questions for *Tekanan* (Stress)  

## Notes
- Both datasets are used for analysis in the R scripts.
- Data have been cleaned and anonymized before analysis.
- No personally identifiable information is included.
- Variable names are retained in Malay to preserve the original dataset structure.
